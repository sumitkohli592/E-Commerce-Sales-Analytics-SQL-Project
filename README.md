# 📊 E-Commerce Sales Analytics Using Microsoft SQL Server

## Project Overview

This project demonstrates how SQL can be used to solve real business problems using an E-Commerce dataset.

The project contains a complete database design, data analysis queries, and business insights generated using Microsoft SQL Server.

It covers Sales Analysis, Customer Analysis, Inventory Management, Shipping Performance, Payment Analysis, Returns Analysis, and Business Intelligence.
